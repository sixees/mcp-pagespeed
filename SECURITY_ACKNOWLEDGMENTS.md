## How to Get Listed

If you've responsibly disclosed a security vulnerability following our [Security Policy](SECURITY.md), you may be
eligible for recognition here. We will:

- Add your name (or handle) and a link to your profile/website
- Note the date of disclosure
- Provide a brief, non-sensitive description of the vulnerability class
- Include the severity level

If you prefer to remain anonymous, let us know when reporting.
